# Internal Planning Docs

Working documents for maintainers: implementation trackers, proposals, and
roadmap notes that are not ready for public consumption. These are kept out
of `docs/` on purpose — everything under `docs/` is built and published by
the VitePress docs site, and VitePress renders every `.md` file it finds
there (linked in the nav or not).

Before the repository goes public, review this folder and decide, file by
file, whether it should stay private, move into `docs/` as a public page, or
be deleted once the plan it describes has landed.
